# HOUSING POLICY ANALYSIS WEBSITE
This website is designed to allow users to search federal, state and city legislation and ask questions about the contents.  It will also be designed to maintain reviews from professionals in the area of these laws and proposals.

It is still in the design phase, so we will start with some database work and an API server to start.

## DATA RETRIEVAL
The first step will be to write an API that can be called to pull an individual law from api.congress.gov.
